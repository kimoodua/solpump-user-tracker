    <td align="center" style="background-color: #0E3049; border-radius: 10px; padding: 20px;">
      <h3 align="center">❓ Is this extension safe?</h3>
      <p align="center">Yes! The extension only reads public data from SolPump.com. It doesn't access your wallet, passwords, or any sensitive information. All code is open source for transparency.</p>
      <br>
      <h3 align="center">❓ Does it slow down my browser?</h3>
      <p align="center">No. The extension uses minimal resources with efficient caching and only runs on SolPump.com pages.</p>
      <br>
      <h3 align="center">❓ How many users can I track?</h3>
      <p align="center">Unlimited! Track as many users as you want. The extension uses smart caching to handle large lists efficiently.</p>
      <br>
      <h3 align="center">❓ Can I export my user list?</h3>
      <p align="center">Your user list is stored locally in your browser. You can manually create a backup .txt file with usernames (one per line).</p>
      <br>
      <h3 align="center">❓ Why aren't some users being detected?</h3>
      <p align="center">The extension only detects users who appear in the active bet list. If a user isn't currently in a game, they won't be shown as "found".</p>
    </td>
  </tr>
